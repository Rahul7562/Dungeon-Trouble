# Core Engine Library

The Core Engine Library is the foundational layer of DungeonTrouble. It provides basic utilities, timing mechanisms, system types, and the application lifecycle architecture.

It holds strictly zero dependencies on external APIs (excluding basic C++ standard libraries and OS detection macros) to ensure it can compile rapidly and integrate cleanly across Server, Client, and Editor applications.

See `Architecture.md` and `PublicAPI.md` for more details.
