# Roadmap

## Phase 1: Core Engine Foundation (Milestones 1-50)
* M1-M10: Core types, memory allocators, logging, and build system.
* M11-M20: Math library, Platform layer, Windowing, and Input.
* M21-M30: Renderer abstraction, basic 2D drawing, Shader pipeline.
* M31-M40: Resource manager, VFS (Virtual File System).
* M41-M50: ECS foundation, basic scene management.

## Phase 2: Game Framework & Tools (Milestones 51-100)
* M51-M60: Editor UI framework, gizmos, property inspectors.
* M61-M70: Physics engine (2D collision, triggers).
* M71-M80: Audio system (FMOD/SoLoud integration).
* M81-M90: Animation system (sprite sheets, blending).
* M91-M100: Scripting integration and initial tooling.

## Phase 3: Multiplayer & Networking (Milestones 101-150)
* M101-M110: Networking core (Sockets, Reliable UDP).
* M111-M120: Server architecture, headless mode.
* M121-M130: Entity replication, state synchronization.
* M131-M140: Client prediction, server reconciliation.
* M141-M150: Chat, authentication, basic MMO server lobby.

## Phase 4: DungeonTrouble Alpha Features (Milestones 151-200)
* M151-M160: Combat foundation, hitboxes, damage numbers.
* M161-M170: Class implementations, skill trees.
* M171-M180: Inventory, items, equipment UI.
* M181-M190: Basic NPC AI, pathfinding (NavMesh/A*).
* M191-M200: First playable dungeon, boss mechanics.

## Phase 5: Beta & Polish (Milestones 201-250+)
* M201-M220: Crafting, economy, trading systems.
* M221-M240: Guilds, social features, housing.
* M241-M250: Performance profiling, optimization, memory audits.
* Post-M250: Live ops, expansion planning.
